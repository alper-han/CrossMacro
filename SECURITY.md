# Security Policy

## Supported Versions

CrossMacro currently provides security updates for:

- `main` branch (always)
- latest stable release (always)

Older releases may receive fixes on a best-effort basis and should not be considered fully supported.

## Reporting a Vulnerability

We take security seriously. If you discover a vulnerability, please do NOT report it as a public issue.

Private vulnerability reporting is enabled for this repository. Please use the [Private Reporting](https://github.com/alper-han/CrossMacro/security/advisories/new) feature to submit vulnerabilities securely.

Alternatively, you can report it by emailing us directly at **crossmacro@zynix.net**.

We will review your report and respond as soon as possible.
